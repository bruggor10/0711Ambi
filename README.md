# sh|ft ensemble – #Order0711  
3D Design Files & Konfigurationen

Dieses Repository enthält die 3D-Design-Dateien, Konfigurationsdateien und weitere technische Ressourcen  für das Projekt **#Order0711** von **sh|ft ensemble**
Die Inhalte dienen der Dokumentation, Weiterentwicklung und Reproduzierbarkeit des Projekts.

---

## Inhalte

### 3D Design Files
Enthält 3D-druckbare Modelle, die im Rahmen von **#Order0711** erstellt wurden.  

### Konfigurationsdateien
Konfigurations- und Einstellungsdateien für:
- AllRA Encoder (IEM Plugins)
- Excel-Tabelle zur Errechnung der Rohrlänge (abhängig vom Radius der Konstruktioin)
